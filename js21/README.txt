  S21 TRACKER — Chrome Extension
  Трекер участников проектов School 21
════════════════════════════════════════════════════════


── УСТАНОВКА ───────────────────────────────────────────

1. Распакуй архив s21-tracker.zip в любую папку
   (например: C:\Users\ИМЯ\s21-tracker\)

2. Открой Google Chrome и перейди по адресу:
   chrome://extensions/

3. Включи «Режим разработчика»
   (переключатель в правом верхнем углу)

4. Нажми «Загрузить распакованное расширение»

5. Выбери папку s21-tracker (ту, где лежит manifest.json)

6. Расширение появится в списке и в панели браузера
   Иконка S21 — в правом верхнем углу Chrome


── АВТОРИЗАЦИЯ ─────────────────────────────────────────

При первом открытии введи логин и пароль от
платформы edu.21-school.ru и нажми «→ ВОЙТИ».

Данные сохраняются локально — при следующем открытии
авторизация произойдёт автоматически.

Для смены аккаунта нажми «↩ ВЫЙТИ» рядом с именем.


── ЧТО УМЕЕТ ───────────────────────────────────────────

▸ ПОИСК УЧАСТНИКОВ
  Введи ID проекта или курса — расширение подтянет
  название и описание. Выбери кампусы и статусы
  проекта, нажми «▶ НАЙТИ УЧАСТНИКОВ».

  Для каждого участника отображается:
  • Логин, кампус, уровень, XP
  • Статус проекта (ACCEPTED, IN_PROGRESS и т.д.)
  • Статус участника (ACTIVE, FROZEN и т.д.)

▸ ФИЛЬТРЫ
  Кампусы: все / MSK+KZN+NSK / остальные / вручную
  Статусы проекта: любая комбинация из 6 вариантов
  Поиск по логину в найденном списке

▸ СОРТИРОВКА
  Клик по заголовку колонки — сортировка по ней,
  повторный клик — обратный порядок

▸ ДЕТАЛЬНАЯ КАРТОЧКА
  Клик по строке участника открывает панель с
  подробностями и ссылкой на профиль

▸ СОХРАНЕНИЕ СПИСКОВ
  «↓ СОХРАНИТЬ СПИСОК» — сохраняет текущих
  найденных участников как точку отсчёта.

  При следующем поиске новые участники (которых
  не было в сохранённом списке) помечаются
  зелёным значком NEW.

▸ ОБЗОР ПРОЕКТОВ ПРИ ЗАПУСКЕ
  При открытии расширения автоматически
  загружаются все сохранённые проекты с
  актуальными данными — сразу видно сколько
  участников и сколько новых появилось.

  Клик «Открыть →» переходит к полному поиску
  с сохранёнными настройками этого проекта.

  Клик «✕» удаляет проект из обзора.

▸ ЭКСПОРТ
  «⇩ ЭКСПОРТ CSV» — скачивает найденных
  участников в файл для Excel или Google Sheets

▸ НАСТРОЙКИ ХРАНЯТСЯ ЛОКАЛЬНО
  Всё (логин, пароль, списки участников, настройки
  фильтров) хранится только на твоём компьютере
  в хранилище браузера. Никуда не отправляется.


── ВОЗМОЖНЫЕ ПРОБЛЕМЫ ──────────────────────────────────

«Авторизация не проходит»
  → Проверь логин и пароль на edu.21-school.ru
  → Убедись что расширение установлено правильно
    (в chrome://extensions/ статус «Включено»)

«Нет данных / пусто»
  → Возможно истёк токен — нажми «↩ ВЫЙТИ»
    и войди заново

«Загружается очень долго»
  → API School 21 ограничивает количество запросов.
    Расширение автоматически повторяет запросы
    при ошибке 429 — просто подожди.


── ДАННЫЕ И ПРИВАТНОСТЬ ────────────────────────────────

Расширение не имеет серверной части.
Все данные хранятся локально в chrome.storage.local.
Запросы идут напрямую к официальному API School 21.
