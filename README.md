# S21 Tracker

Инструмент для отслеживания участников проектов [School 21](https://21-school.ru).  
Позволяет находить новых участников, фильтровать по кампусам и статусам, сохранять списки и сравнивать результаты.

## C# версия

### Требования

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

### Установка и запуск

```bash
# Клонируй репозиторий или скачай файлы
git clone <repo-url>
cd <repo-dir>

# Создай файл с учётными данными рядом с Program.cs
# Формат: три строки — логин, пароль, порт
echo "твой-логин
твой-пароль
8080" > credentials.txt

# Запусти
dotnet run
```

После запуска откроется браузер на `http://localhost:8080/`.

### Структура файлов

```
Program.cs              — C# сервер (HTTP прокси + раздача HTML)
school21-tracker.html   — весь UI (кладётся рядом с Program.cs)
cs21.csproj             — проект .NET
credentials.txt         — логин/пароль/порт (создаётся вручную)
saved_*.json            — сохранённые списки участников (создаются автоматически)
settings_*.json         — настройки фильтров по проектам (создаются автоматически)
```

### Как работает

Сервер запускается на `localhost:8080` и выполняет две функции:

1. **Раздаёт** `school21-tracker.html` как веб-страницу
2. **Проксирует** запросы к API School 21 — решает проблему CORS

Все данные хранятся локально в JSON файлах рядом с `Program.cs`.

## Chrome Extension

### Установка

1. Скачай и распакуй архив `s21-tracker.zip`
2. Открой Chrome → `chrome://extensions/`
3. Включи **«Режим разработчика»** (правый верхний угол)
4. Нажми **«Загрузить распакованное расширение»**
5. Выбери папку `s21-tracker` (ту, где лежит `manifest.json`)
6. Иконка S21 появится в панели браузера

### Структура файлов расширения

```
s21-tracker/
├── manifest.json       — конфигурация расширения (MV3)
├── popup.html          — интерфейс
├── popup.js            — вся логика
├── background.js       — service worker (keep-alive, CORS)
├── rules.json          — declarativeNetRequest (подстановка origin)
├── icon16/48/128.png   — иконки
└── README.txt          — краткая инструкция
```

### Как работает

- Запросы к API School 21 идут **напрямую из popup** — браузер разрешает это благодаря `host_permissions` в манифесте
- Заголовок `origin` для авторизации подставляется через `declarativeNetRequest`
- Данные хранятся в `chrome.storage.local` — только на твоём компьютере

## Функциональность (обе версии)

### Поиск участников
- Введи **Project ID** (число из URL проекта на платформе)
- Выбери **кампусы**: все / MSK+KZN+NSK / остальные / вручную
- Выбери **статусы проекта**: ACCEPTED, IN_PROGRESS, IN_REVIEWS, REGISTERED, ASSIGNED, FAILED
- Нажми **▶ НАЙТИ УЧАСТНИКОВ**

Для каждого участника отображается: логин, кампус, уровень, XP, статус проекта, статус участника.

### Сохранение и отслеживание новых
- **↓ СОХРАНИТЬ СПИСОК** — сохраняет текущих найденных участников как точку отсчёта
- При следующем поиске новые участники (которых не было раньше) помечаются зелёным **NEW**
- **Обзор при запуске** — автоматически загружает все сохранённые проекты с актуальными данными

### Экспорт
- **⇩ ЭКСПОРТ CSV** — для Excel / Google Sheets
- **⇩ ЭКСПОРТ HTML** — самодостаточная страница в том же тёмном стиле

### Дополнительно
- Клик по строке участника — детальная карточка + ссылка на профиль
- Сортировка по любой колонке (клик по заголовку)
- Поиск по логину в найденном списке

## API School 21

Используется официальный API платформы. Документация: `Sctool21_OpenAPI.json`.

**Авторизация:** Keycloak (`grant_type=password`, `client_id=s21-open-api`)

**Base URL:** `https://platform.21-school.ru/services/21-school/api`

Коллекция Postman для тестирования: `School21_postman_collection.json`
