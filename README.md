# S21 Tracker

Инструмент для отслеживания участников проектов [School 21](https://platform.21-school.ru/).
Позволяет находить новых участников, фильтровать по кампусам и статусам, сохранять списки и сравнивать результаты.

# Установка и запуск

```bash
# Клонируй репозиторий
git clone <repo-url>
cd <repo-dir>

# Настрой файл конфигурации
```

Настрой файл **`appsettings.json`** рядом с `Program.cs`.

Пример:

```json
{
  "School21": {
    "Username": "твой-логин",
    "Password": "твой-пароль",
    "Port": 2121
  }
}
```

После этого запусти сервер:

```bash
dotnet run
```

После запуска откроется браузер:

```
http://localhost:{Port}/
```

# Как работает C# версия

Сервер запускается на `localhost` и выполняет две задачи:

1. **Раздаёт** `school21-tracker.html` как веб-интерфейс
2. **Проксирует** запросы к API School 21 (решает проблему CORS)

Все данные хранятся локально в JSON файлах на компьютере пользователя.

# Chrome Extension

## Установка

1. Скопируй папку `js21` туда где её никто не тронет
2. Открой Chrome → `chrome://extensions/`
3. Включи **Режим разработчика**
4. Нажми **Загрузить распакованное расширение**
5. Выбери папку `js21` (где лежит `manifest.json`)
6. Иконка S21 появится в панели браузера

# Как работает расширение

* Запросы к API School 21 идут **напрямую из popup**
* Заголовок `origin` для авторизации подставляется через `declarativeNetRequest`
* Данные хранятся в `chrome.storage.local`

# Функциональность

## Поиск участников

1. Введи **Project ID** (число из URL проекта)
2. Выбери **кампусы**
3. Выбери **статусы проекта**
4. Нажми **▶ НАЙТИ УЧАСТНИКОВ**

Для каждого участника отображается:

* логин
* кампус
* уровень
* XP
* статус проекта
* статус участника

# Отслеживание новых участников

* **↓ СОХРАНИТЬ СПИСОК** — сохраняет текущий список как базовый
* При следующем поиске новые участники помечаются **NEW**
* **Обзор при запуске** — загружает все сохранённые проекты

# API School 21

Используется официальный API платформы.

**Авторизация:**
Keycloak (`grant_type=password`, `client_id=s21-open-api`)

**Base URL**

```
https://platform.21-school.ru/services/21-school/api
```

Документация: `Sctool21_OpenAPI.json`
Postman коллекция: `School21_postman_collection.json`

---

# Данные и приватность

* Логин и пароль хранятся **только локально**
* Никакие данные не отправляются на сторонние серверы
* Все запросы идут напрямую к официальному API School 21
